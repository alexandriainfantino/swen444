## Installation
Install pip (may need to run with `sudo`)
```
easy_install pip
```

Install Flask (may need to run with `sudo`)
```
pip install Flask
```

Install the project
```
git clone git@github.com:alexandriainfantino/swen444.git
```
or
```
git clone https://github.com/alexandriainfantino/swen444.git
```
